# Rails Syntax Highlighting for Visual Studio Code
Extends Visual Studio Code's Ruby Language Grammars with Rails Library.

Syntax Identification for Rails Library using [Rails API](https://api.rubyonrails.org).

## Mappings List:
### ActionText
